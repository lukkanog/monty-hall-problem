## Como executar em ambiente de desenvolvimento local
- Clone esse repositório;
- Instale as dependências do projeto, através do comando:
``` bash
# Utilizando npm:
npm install

# Utilizando yarn:
yarn
```

- Inicie o servidor web de desenvolvimento local:
``` bash
# Utilizando npm:
npm run serve

# Utilizando yarn:
yarn serve
```
